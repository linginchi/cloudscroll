## 二、頁面流程

```
封面 →（點擊翻頁）→ 書架 →（點擊書影）→ 目錄 →（點擊文章）→ 閱讀頁（翻頁）
```

### 2.1 封面頁

- **背景**：多層 CSS/SVG，武夷山水、九曲溪、漁翁竹排剪影、斜陽、雲舒雲卷、人字大雁北飛動畫
  - 實現方式：多層 `<div>` 疊加，每層獨立的 CSS animation（parallax 速率）
  - 遠山層：淡墨 SVG path，極慢水平偏移
  - 中景層：武夷山巒剪影，淡青淡赭點綴
  - 溪流層：九曲溪曲線，水面微光
  - 人物剪影：漁翁撐竹排 + 夫婦坐排首，純色剪影
  - 雲層：透明度呼吸動畫（`opacity` 3-5s loop）
  - 雁群：人字排列的 SVG `<path>`，CSS `translateX` 緩慢橫移（30-60s 一個循環）
  - 所有圖層使用純 CSS，不依賴 Canvas 或 WebGL
- **金句**：大字居中，Noto Serif SC Light（300）
  > 九曲溪頭，竹排一葉。行到水窮處，坐看雲起時。
- **網站名**：「雲箋文舍 · Cloudscroll」置於頁面底部小字
- **無按鈕、無書影**。使用者點擊畫面任一處觸發翻頁過渡到書架頁

### 2.2 書架頁

- 展示書本縮略圖：《我的人生旅行》
- 書影卡片：米白封面、仿古裝幀線、書名楷體
- 底部提示文字「翻開書頁 →」
- 點擊書影卡片 → 翻頁進入目錄

### 2.3 目錄頁

- Header：「《我的人生旅行》目錄 / 林樺」
- 文章列表，含約 10 篇遊記標題
- Footer：「目錄 · i」
- 點擊任一文章標題 → 進入該文章之閱讀頁

### 2.4 閱讀頁

- Header：文章標題 + 作者名
- 正文區域：Noto Serif SC Regular，適合長文閱讀的行距與字級
- Footer：「《我的人生旅行》· pN」
- 手指從右往左滑動觸發橫向翻頁動畫
  - CSS 3D transform 實現書頁捲起翻開效果
  - 翻頁過程中可見「紙背面」微透效果
- 語言切換：繁體中文 / English 雙語

## 三、通用頁面框架

每頁統一使用邊框結構：

```
┌──  ──  ──  ──  ──  ──  ──  ──  ──┐
│  ┌───────────────────────────┐  │
│  │  [Header: 文章標題 / 林樺] │  │
│  │  ─────────────────────── │  │
│  │                           │  │
│  │      內容區域              │  │
│  │                           │  │
│  │  ─────────────────────── │  │
│  │  [Footer: 書名 · 頁碼]    │  │
│  └───────────────────────────┘  │
└──  ──  ──  ──  ──  ──  ──  ──  ──┘
```

- 外邊框：2px 暖棕色細線（`#c4a882`），內縮 16px
- Header：文章標題用 Noto Serif SC SemiBold，作者名小字
- Footer：書名 + 頁碼，細線分隔

## 四、視覺風格

### 4.1 配色

| 用途 | 色值 | 說明 |
|------|------|------|
| 頁面底色 | `#f5f0e8` | 米白宣紙色 |
| 主要文字 | `#3d2b1f` | 深棕，低對比暖色 |
| 次要文字 | `#8b7355` | 暖灰棕 |
| 邊框 / 裝飾線 | `#c4a882` | 暖棕線條 |
| 卡片底色 | `#faf7f2` | 較亮的米白 |
| 極淡文字 | `#a09080` | 頁碼、簽名等 |

### 4.2 字體

| 層級 | 字體 | 字重 | 用途 |
|------|------|------|------|
| 金句 | Noto Serif TC | Light (300) | 封面大字 |
| 書名 | Noto Serif TC | SemiBold (600) | 書影卡片 |
| 標題 | Noto Serif TC | SemiBold (600) | Header 文章名 |
| 正文 | Noto Serif TC | Regular (400) | 閱讀頁內文 |
| 小字 | Noto Serif TC | Light (300) | 頁碼、簽名 |

- 繁體中文使用 `Noto Serif TC`（思源宋體 台灣版本）
- 英文版本使用 `Noto Serif`（拉丁版本）配 `Noto Serif TC`
- CSS font-family 回退鏈：`'Noto Serif TC', 'Noto Serif SC', 'Songti SC', 'PMingLiU', '新細明體', serif`
- 中文字體做子集化（subset），只提取網站用到的字，WOFF2 壓縮至數十 KB
- 子集化工具：`glyphhanger` 或 `fonttools pyftsubset`

### 4.3 金句語氣

蘇東坡式的豁達灑脫——短句、自然意象、禪意不做作。金句可隨文章意境更換，輪換備選：

- 「山不動，雲在走。人不語，雁正飛。世間事，何必急。」
- 「這溪水從哪裡來，到哪裡去，誰也不知道。但此刻夕陽落在槳上，就夠了。」

## 六、技術架構

### 6.1 靜態網站 + 極簡 JS

- 純 HTML + CSS + 少量 Vanilla JS（翻頁動畫用）
- 不引入前端框架（React / Vue），保持極簡和加載速度
- 翻頁動畫使用 CSS 3D transform + touch event

### 5.2 部署

| 層 | 工具 |
|----|------|
| 代碼託管 | GitHub |
| 構建 / 部署 | Cloudflare Pages（連接 GitHub 自動部署） |
| 域名 | `cloudscroll.net`（Cloudflare DNS） |

### 5.3 字體子集化

- 使用 `fonttools`（Python）或 `glyphhanger` 進行字體子集化
- 構建時自動提取 HTML 中用到的全部中文字，生成最小化 WOFF2
